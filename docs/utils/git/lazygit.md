# lazygit

```bash
echo "alias lg='lazygit'" >> ~/.zshrc
```

? 查看所有快捷键

## 快捷键

- c commit
- p pull
- P push
- q quit
- r 更改commit内容
- space 空格 add 单个文件
- space 切换分支
- i 可以将文件添加到.gitIgnore中
- e 打开该文件

: 可以执行shell命令

## problem

- 移除所有已被跟踪的 .DS_Store 文件
  git rm -r --cached .DS_Store

## branch

- 新建分支 3 n branch_name
- 切换分支 space

## cherry-pick

当前分支 4 panel - shift+c
切换为另外一个分支  4-panel-shift+v
